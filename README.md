## 仿Antd组件库
eason-ui 组件库是参考 antd 组件库设计， 算是笔者的一次组件库设计尝试。<br>
<br>
技术栈：React Hooks + Typescript + axios + scss + jest
<br>
下载依赖：
```
npm install
```
启动Story book
```
npm run storyBook
```
也可以在自己的项目中使用下面的命令下载组件库
```
npm i easons-ui
```
组件库在线文档:https://vermillion-tiramisu-90ecac.netlify.app/?path=/story/introduction--page
